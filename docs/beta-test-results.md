# Результаты бета-тестирования TalkPro

- **Дата проведения:** 20 апреля – 15 мая 2026
- **Количество участников:** 16 рекрутеров и HR-специалистов

## Ключевые показатели

| Метрика | Значение |
|---------|----------|
| Успешность создания событий в календаре | 100% (16/16) |
| Удобство интерфейса (средняя оценка) | 8.63/10 |
| Готовность рекомендовать (NPS) | 56.2% (промоутеры 9–10 – 56.2%) |
| Средняя вероятность ИИ (по оценке системы) | 44.3% |
| Согласие с оценкой детекции ИИ (Да) | 56.2% (9/16) |
| Частичное согласие с детекцией ИИ | 43.8% (7/16) |
| Средняя оценка точности подсветки проблемных мест | 7.0/10 |
| Средняя оценка полезности вопросов для собеседования | 6.13/10 |
| Средняя экономия времени на холодном поиске | в 3–6 раз быстрее ручного |

## Основные выводы
- Пользователи высоко оценили возможность быстрого холодного поиска и интеграцию с календарём.
- Необходимо добавить расширенные фильтры (возраст, пол, страна, готовность к переездам, формат работы).
- Требуется интеграция с другими job-сайтами (hh.ru, Авито, Работа России).
- Функция детекции ИИ требует улучшения интерпретации результатов и контекстной подсветки.

Полные данные доступны в личной папке проекта и в отчете ссылка "Результаты бета-тестирования"
