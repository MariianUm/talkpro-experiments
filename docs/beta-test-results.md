# Результаты бета-тестирования TalkPro

- **Дата проведения:** 20 апреля – 15 мая 2026
- **Количество участников:** 16 рекрутеров и HR-специалистов

## Ключевые показатели

| Метрика | Значение |
|---------|----------|
| Точность детекции ИИ (субъективная) | 44% средняя вероятность (согласны 62.5%) |
| Успешность создания событий в календаре | 100% |
| Удобство интерфейса (средняя оценка) | 8.7/10 |
| Готовность рекомендовать (NPS) | 56.25% |
| Средняя экономия времени на холодном поиске | в 3–6 раз быстрее ручного |

## Основные выводы
- Пользователи высоко оценили возможность быстрого холодного поиска и интеграцию с календарём.
- Необходимо добавить расширенные фильтры (возраст, пол, страна, готовность к переездам).
- Требуется интеграция с другими job-сайтами (hh.ru, Авито, Работа России).

Полные данные и анкеты доступны в [личной папке проекта].
