# Сценарий бета-тестирования TalkPro

## Предварительные шаги
1. Регистрация/вход в систему (email: user1@example.com, пароль: user123).
2. Ознакомление с интерфейсом (главная страница, меню).

## Основные сценарии

### Сценарий 1: Загрузка резюме и анализ
- Перейти на страницу «Загрузить резюме».
- Загрузить файл резюме (PDF, DOCX).
- Запустить анализ.
- Оценить:
  - точность детекции ИИ (сравнить с субъективным мнением),
  - полезность подсветки проблемных мест,
  - качество сгенерированных вопросов для собеседования.

### Сценарий 2: Холодный поиск кандидатов
- Перейти на страницу «Холодный поиск».
- Ввести ключевые слова (например, «Python разработчик»).
- Выбрать город, опыт, желаемый доход (опционально).
- Оценить релевантность найденных кандидатов и скорость поиска.

### Сценарий 3: Создание события в календаре
- Выбрать кандидата из результатов поиска или списка резюме.
- Перейти к созданию встречи.
- Указать дату, время, email участников.
- Убедиться, что событие появилось в календаре.

## Оценка
- Каждый сценарий оценивается по шкале 1–10 (где 1 – очень плохо, 10 – отлично).
- Открытые вопросы:
  - Что понравилось?
  - Что не понравилось / раздражало?
  - Каких функций не хватает?
- NPS: «Какова вероятность, что вы порекомендуете TalkPro коллегам?» (0–10).

## Инструкция для тестера
- Длительность тестирования: 20–30 минут.
- Заполнить анкету после каждого сценария.
- Запись экрана приветствуется (по желанию).

Полный сценарий в личной папке проекта и в Отчете ссылка на файл "Сценарии бета-тестирования"
