# TalkPro Experiments

Этот репозиторий содержит код и артефакты экспериментов по проверке технологических гипотез проекта TalkPro – умного помощника рекрутера.

## Структура репозитория

- `talkpro-ai-services/` – код AI-сервиса (шлюз GigaChat, адаптер SuperJob, клиент календаря)
- `tests/` – скрипты нагрузочного и интеграционного тестирования
- `docs/` – документация (инструкция по развёртыванию, результаты бета-тестирования, сценарии)
- `DEPLOYMENT.md` – инструкция по развёртыванию и обновлению на сервере

## Эксперименты

### Гипотеза H1 (AI Gateway)
- **Цель:** снизить P95 latency на 25% и сократить платные запросы к GigaChat на 35%
- **Результаты:** экономия платных запросов 30.7% (близко к цели 35%), P95 latency увеличилась (требуется оптимизация)
- **Код:** `tests/load_test_gateway.py`, `load_test_output.txt`
- **Статус:** частично подтверждена

### Гипотеза H2 (Calendar Service)
- **Цель:** обеспечить успешность создания событий календаря ≥99.5% при 10% сбоев
- **Результаты:** 100% успешности при 10% сбоев, время подтверждения <0.01 с
- **Код:** `tests/test_hypothesis2.py`, `yandex_calendar/`
- **Статус:** полностью подтверждена

### Гипотеза H3 (SuperJob Adapter)
- **Цель:** успешность получения данных ≥99% со средней скоростью ≤15 с
- **Результаты:** 100% успешности, среднее время 0.05 с, P95 0.28 с
- **Код:** `job_search/`
- **Статус:** полностью подтверждена
